WOD Tracker - Programa de Crossfit

A Ferramenta de Crossfit foi feita para a disciplina de Fundamentos de Programação com mais 4 colegas, onde o objetivo era criar um programa automatizado para montar e mostrar estatísticas de treinos de crossfit.

O programa roda em qualquer terminal e é possível adicionar treinos, visualizar treinos, ver estatísticas e até mesmo receber medalhas quando alguma meta for concluída. Tudo é editável, desde as metas já criadas, até o novos treinos.

Foi feito utilizando Python. O código está disponível no GitHub
